# RMD decor

`RMD decor` — учебный full-stack проект интернет-магазина товаров для домашнего декора. В проекте реализованы каталог товаров, карточка товара, корзина, оформление заказа, личный кабинет с редактированием данных, история заказов пользователя, вход администратора, просмотр всех заказов, статистика продаж и базовая аналитика посещаемости.

## Технологии

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- PostgreSQL

## Настройка базы данных

1. Создайте базу данных PostgreSQL с именем `decor_store`.
2. Выполните SQL-файлы в таком порядке:

```sql
\i db/schema.sql
\i db/seed.sql
```

Если база уже была создана раньше и нужно просто перевести существующие данные на русский и обновить цены/изображения, выполните:

```sql
\i db/update_to_russian.sql
```

## Настройка `.env`

Создайте файл `.env` рядом с `package.json` и укажите:

```env
PORT=3000
DB_HOST=localhost
DB_PORT=5432
DB_NAME=decor_store
DB_USER=postgres
DB_PASSWORD=ваш_пароль
ADMIN_LOGIN=admin
ADMIN_PASSWORD=admin123
ADMIN_TOKEN=rmd-admin-access
```

## Установка и запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите сервер:

```bash
npm run dev
```

или:

```bash
npm start
```

3. Откройте в браузере:

[http://localhost:3000](http://localhost:3000)
